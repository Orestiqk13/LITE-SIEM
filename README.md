## Περιγραφή (Overview)
Το **LITE-SIEM** είναι ένα ελαφρύ, ανοικτού κώδικα σύστημα διαχείρισης πληροφοριών και συμβάντων ασφαλείας, σχεδιασμένο ειδικά για περιβάλλοντα περιορισμένων πόρων (IoT, Edge συσκευές, μικρομεσαίες υποδομές). Αντιμετωπίζει το πρόβλημα της τεράστιας κατανάλωσης πόρων (minimal footprint) που χαρακτηρίζει τις εμπορικές λύσεις, διατηρώντας παράλληλα υψηλή ακρίβεια στην ανίχνευση.

Η αρχιτεκτονική του βασίζεται σε τρία επίπεδα:
1. **C Collector:** Ένας εξαιρετικά ελαφρύς πράκτορας συλλογής γεγονότων (event-driven μέσω `inotify`), με στατική δέσμευση μνήμης.
2. **Python Engine:** Η κεντρική μηχανή συσχέτισης (Correlation Engine) η οποία εμπλουτίζει τα γεγονότα με τακτικές και τεχνικές του **MITRE ATT&CK** και αξιολογεί alerts από τον αισθητήρα **Suricata**. Αξιοποιεί επικοινωνία μέσω Unix Domain Sockets (IPC) για μέγιστη ταχύτητα.
3. **Streamlit Dashboard:** Το διαδραστικό γραφικό περιβάλλον για την οπτικοποίηση των περιστατικών σε πραγματικό χρόνο.

## Αρχιτεκτονική του Αποθετηρίου (Repository Structure)
* `collector/` : Ο πηγαίος κώδικας σε γλώσσα C για τη συλλογή OS Logs.
* `engine/` : Τα σενάρια Python για την επεξεργασία, την αντιστοίχιση σε O(1) χρόνο και τη βάση δεδομένων (SQLite).
* `dashboard/` : Τα αρχεία του Streamlit για το UI.
* `config/` : Αρχεία ρυθμίσεων και το στοχευμένο Rulebook (COMMAND_INTEL & Suricata Mapping).

## Οδηγίες Εγκατάστασης (Setup Guide)
*Οι αναλυτικές οδηγίες αρχικοποίησης, εγκατάστασης εξαρτήσεων και εκκίνησης των επιμέρους υποσυστημάτων περιγράφονται αναλυτικά στο Παράρτημα Α του πλήρους κειμένου της Διπλωματικής Εργασίας.*

Για γρήγορη εκκίνηση του εργαστηριακού περιβάλλοντος (απαιτείται Ubuntu 24.04 LTS):
```bash
# 1. Εκκίνηση δικτυακού αισθητήρα
sudo suricata -i enp0s8

# 2. Εκκίνηση πράκτορα συλλογής
sudo ./collector

# 3. Εκκίνηση μηχανής ανάλυσης
python3 engine.py

# 4. Εκκίνηση διεπαφής χρήστη
streamlit run app.py
